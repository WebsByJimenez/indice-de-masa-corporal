# Tu Salud Digital - Calculadora de IMC 🧮🩺

Una calculadora interactiva y moderna de **Índice de Masa Corporal (IMC)** diseñada para ayudar a los usuarios a monitorear su estado nutricional de forma rápida y sencilla. Cuenta con un diseño adaptativo, indicador visual y consejos personalizados según el resultado.

## 🚀 Características

- **Cálculo Preciso:** Obtén tu IMC ingresando género, altura (en cm) y peso (en kg).
- **Indicador Visual (Gauge):** Un medidor gráfico dinámico que ubica visualmente tu resultado entre los rangos (Bajo, Normal, Sobrepeso, Obesidad).
- **Feedback Personalizado:** Muestra el estado actual y proporciona recomendaciones adaptadas en tiempo real.
- **Diseño Adaptable (Responsive):** Interfaz optimizada para su correcto funcionamiento tanto en dispositivos móviles como en ordenadores.
- **Código Semántico y Accesible:** Estructurado siguiendo las mejores prácticas de HTML5 y accesibilidad web.

## 🛠️ Tecnologías utilizadas

- **HTML5:** Estructura semántica del sitio.
- **CSS3:** Estilos personalizados, tipografía (Poppins) y diseño adaptativo.
- **JavaScript (ES6):** Lógica para el cálculo del IMC, manipulación del DOM y animación del indicador visual.
