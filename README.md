# 🧮 Tu Salud Digital — Calculadora de IMC 🩺

**Tu Salud Digital** es una herramienta web interactiva, moderna y accesible diseñada para calcular el Índice de Masa Corporal (IMC) de forma rápida y precisa. Proporciona una clasificación visual clara sobre el estado nutricional del usuario con una experiencia de uso intuitiva y fluida.

## ✨ Características Principales

- **Cálculo Preciso e Instantáneo:** Procesa la relación entre el peso y la altura en tiempo real.
- **Indicador Visual de Salud:** Retroalimentación visual inmediata con colores según la categoría del IMC (Bajo peso, Peso normal, Sobrepeso, Obesidad).
- **Diseño Adaptativo (Responsive):** Interfaz limpia y optimizada para cualquier pantalla (dispositivos móviles, tablets y ordenadores).
- **Validación de Datos:** Evita datos vacíos o valores negativos para garantizar un resultado confiable.
- **Experiencia Accesible:** Estructura de código semántica pensada en la usabilidad y legibilidad.

---

## 🛠️ Tecnologías Utilizadas

- **HTML5:** Estructura semántica del formulario y contenido.
- **CSS3:** Custom Properties (Variables CSS), Flexbox, diseño dinámico y estilos visuales por categoría.
- **JavaScript (ES6+):** Lógica matemática de cálculo, manipulación del DOM y gestión de clases dinámicas.
